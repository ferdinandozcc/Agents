# API Documentation Writer Agent

## What it does

- Parse OpenAPI specs, code, or endpoint descriptions
- Write clear endpoint docs with method, path, params, and examples
- Generate SDK usage examples in Python and JavaScript
- Write authentication and getting-started guides
- Flag undocumented or ambiguous endpoints for review

## Usage

```bash
python3 product/api_documentation_writer/agent.py
```

## Category

`product`

## Tools used

| Tool | Purpose |
|---|---|
| `read_file` | Enables read file capability |
| `write_file` | Enables write file capability |
