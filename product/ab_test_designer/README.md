# A/B Test Designer Agent

## What it does

- Define a clear hypothesis and primary success metric
- Calculate required sample size for statistical power
- Design control and variant conditions clearly
- Define guardrail metrics to prevent regressions
- Interpret results including statistical and practical significance

## Usage

```bash
python3 product/ab_test_designer/agent.py
```
