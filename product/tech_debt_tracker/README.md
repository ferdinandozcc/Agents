# Tech Debt Tracker Agent

## What it does

- Catalog debt items with description, location, and type
- Estimate impact on performance, velocity, and reliability
- Score debt by urgency multiplied by impact
- Recommend a paydown strategy alongside feature work
- Track debt reduction over time to show progress

## Usage

```bash
python3 product/tech_debt_tracker/agent.py
```
