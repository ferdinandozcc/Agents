# Persona Builder Agent

## What it does

- Synthesize qualitative data from interviews, surveys, and tickets
- Identify distinct user archetypes based on behaviors and goals
- Build detailed personas with demographics, goals, and frustrations
- Suggest which persona to prioritize for a given feature
- Keep personas grounded in data rather than assumptions

## Usage

```bash
python3 product/persona_builder/agent.py
```
