# Churn Analysis Agent

## What it does

- Analyze churned vs retained customers for differentiating signals
- Identify the top predictive features of churn behavior
- Build a churn risk scoring model from historical data
- Segment churn by reason: price, product, competition, or lifecycle
- Recommend interventions timed to leading indicators of churn

## Usage

```bash
python3 data/churn_analysis/agent.py
```
