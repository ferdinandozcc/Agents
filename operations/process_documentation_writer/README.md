# Process Documentation Writer Agent

## What it does

- Interview the user about a process through structured questions
- Document the process as a step-by-step SOP
- Include decision points, edge cases, and escalation paths
- Format in a clear, scannable structure with roles and responsibilities
- Suggest process improvements spotted during documentation

## Usage

```bash
python3 operations/process_documentation_writer/agent.py
```

`operations`
