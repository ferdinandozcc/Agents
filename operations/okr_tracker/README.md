# OKR Tracker Agent

## What it does

- Help teams write well-formed Objectives and Key Results
- Track progress against each KR on a 0.0 to 1.0 scale
- Score OKRs at end of quarter using standard methodology
- Flag at-risk KRs needing attention mid-quarter
- Facilitate OKR check-in conversations with the team

## Usage

```bash
python3 operations/okr_tracker/agent.py
```

## Category

`operations`

| Tool | Purpose |
|---|---|
