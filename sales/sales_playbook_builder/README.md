# Sales Playbook Builder Agent

## What it does

- Build competitive battle cards highlighting strengths vs competitors
- Create objection handling guides with proven responses
- Develop discovery question banks by persona and industry
- Write qualification criteria checklists using MEDDIC or BANT
- Define deal stage exit criteria and stage-by-stage playbooks

## Usage

```bash
python3 sales/sales_playbook_builder/agent.py
```

## Category

`sales`

## Tools used

| Tool | Purpose |
|---|---|
| `search_web` | Enables search web capability |
| `read_file` | Enables read file capability |
| `write_file` | Enables write file capability |
