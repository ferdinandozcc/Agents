# Email Copywriter Agent

## What it does

- Write subject lines in 3 variants: curiosity, benefit, and urgency
- Write preview text that complements the subject line
- Write email body with clear hierarchy and one idea per section
- Write CTAs that are action-oriented, specific, and low-friction
- Produce a plain-text version for deliverability

## Usage

```bash
python3 marketing/email_copywriter/agent.py
```
