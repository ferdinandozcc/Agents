# Lead Magnet Creator Agent

## What it does

- Identify high-value problems your ICP wants solved urgently
- Recommend the best format: guide, checklist, template, or calculator
- Write the full outline and content for the chosen lead magnet
- Write landing page copy and thank-you page messaging
- Create an email nurture sequence triggered by the download

## Usage

```bash
python3 marketing/lead_magnet_creator/agent.py
```
