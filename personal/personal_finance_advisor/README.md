# Personal Finance Advisor Agent

## What it does

- Understand financial situation: income, expenses, debts, and savings
- Set and track savings goals over time
- Explain investment basics such as index funds and ETFs
- Suggest debt paydown strategies like avalanche or snowball
- Model different financial scenarios for planning

## Usage

```bash
python3 personal/personal_finance_advisor/agent.py
```

`personal`
